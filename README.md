# English Hash Dictionary 

This is a project from CS230 that uses a hash map that checks words that usere types to see if they are spelled correctly 

## Contents

* `Dictionary.java`
* `EnglishHashDictionary.java` this file contains functions that allows you to add and remove words for a list of words that acts like a dictionary 
* `EnglishHashDictionaryTest.java` this file tests the `EnglishHashDictionary.java` file
* `Speller.java` this file takes text files and checks if each word in the file is spelled correctly and documents how frequently each word appears. 
* `SpellerTest.java` thig file tests the `Speller.java` file
